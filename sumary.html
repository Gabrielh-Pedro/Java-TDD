O que é TDD?

TDD (Desenvolvimento Orientado a Testes) é uma abordagem de desenvolvimento de software em que você escreve os testes antes de escrever o código de produção. Essa prática promove um ciclo de desenvolvimento curto, onde você escreve um teste, faz o teste falhar, escreve o código para passar no teste e, em seguida, refatora o código.

Esses testes são para a aula de Qualidade e Testes de Software.
